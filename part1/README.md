# Part 1 – Data Preparation and Exploratory Data Analysis

## Objective

This project demonstrates data loading, cleaning, exploration, and visualization using the California Housing dataset.

## Dataset

California Housing Dataset from scikit-learn.

## Steps

- Load dataset
- Create categorical feature
- Check missing values
- Display descriptive statistics
- Generate visualizations

## Generated Plots

- House Price Histogram
- Income vs House Price Scatter Plot
- Income Category Bar Chart
- Correlation Heatmap

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python eda.py
```

Plots will be saved inside the `plots/` folder.

## Key Findings

- House price increases with median income.
- Most houses fall into the medium-income category.
- Median income has one of the strongest correlations with house price.
